# R-Tutorials
## Background
R 是做什么的？
R 是一个有着统计分析功能及强大作图功能的软件系统，是由新西兰奥克兰大学统计学系的 Ross Ihaka 和 Robert Gentleman 共同创立。由于 R 受 Becker, Chambers & Wilks 创立的 S和 Sussman 的 Scheme 两种语言的影响，所以 R 看起来 和S 语言非常相似。
